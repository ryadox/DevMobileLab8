>> Les Outils Utilises : 

-XAMPP : stands for Cross-Platform Apache Maria-DB PHP Perl.
-Apache : a Free and OpenSource web server known for its flexibility, reliability, and robust features.
-Maria-DB : is a popular, open-source relational database management system (RDBMS) designed as a high-performance, drop-in replacement for MySQL. 
-PHP : an interpreted scripting language widely-used, open-source server-side scripting language specifically designed for web development .
-Perl : Perl is a general-purpose, high-level interpreted programming language known for its robust text processing capabilities and flexibility.
